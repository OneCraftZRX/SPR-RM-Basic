图像迷你实验室，可进行图像实验，图像处理，图像分析。  



## 食用指南  
- 项目环境：PyQt6 + Qt Designer + Python 3.x  
- ui编译：`python -m PyQt6.uic.pyuic ImageMiniLabUI.ui -o ImageMiniLabUI.py`  
- 包：opencv-python、PyQt6、numpy
